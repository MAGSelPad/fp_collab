# Fundamentos de Programación - Ayudantías

Repositorio con ejercicios y notebooks de práctica para la materia de Fundamentos de Programación.

## Contenido

Cada notebook corresponde a una semana de ayudantía e incluye:

- Ejercicios prácticos
- Soluciones propuestas

## Temas

- Variables y tipos de datos
- Aleatoriedad
- Strings
- Condicionales
- Bucles
- Funciones
- Listas
- Diccionarios
- Archivos


## Cómo usar los notebooks

1. Abrir el notebook correspondiente.
2. Hacer clic en “Open in Colab”.
3. Intente desarrollar la solución del problema en papel.
4. Transcriba su solución en el bloque de código proporcionado.
5. Verifique con las soluciones propuestas.
6. Ejecutar las celdas de código.


## Requisitos

- Cuenta de Google
- Acceso a Google Colab
- Conocimientos vistos en clase teórica
